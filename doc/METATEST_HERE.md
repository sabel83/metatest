# METATEST_HERE

## Synopsis

```cpp
#define METATEST_HERE \
  // unspecified
```

## Description

Macro for constructing a [`location`](location.html) value, representing
the source location that invokes `METATEST_HERE`.

## Header

```cpp
#include <metatest/location.hpp>
```
