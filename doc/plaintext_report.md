# metatest::plaintext_report

## Synopsis

```cpp
bool plaintext_report(std::ostream &os_)
```

## Description

Displays the test results in a plain-text format. Returns `true` when there
was no failed test case.

## Header

```cpp
#include <metatest/plaintext_report.hpp>
```
